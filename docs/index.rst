**************
d-portal
**************

d-portal is a tool which helps you explore International Aid Transparency Initiative (IATI) data.

It provides:

- Views of individual IATI activities.
- A search function.
- Summaries of IATI data by reporting organisation or recipient country.


Open source
------------------

d-portal.org is an open source platform. This means you can use all aspects of the platform to build your own portal using the `free source code <https://github.com/IATI/D-Portal>`_.
